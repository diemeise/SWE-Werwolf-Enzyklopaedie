Single Resposibility Principle

	Klasse	-	-	-	-	-	Resposibility

Plugins

	Kommandos					Mapping Benutzerinput auf Funktionen
	Main						Main halt
	MySQLAuthentifizierung		Authentifizierung an einer MySQL
	MySQLVerbindung				Datenbank spezifische Implementierung des SQLVerbindung Interfaces
	
Adapter

	GameController				Ansteuerung des Gameloops
	OutputAdapter				Erzeugung von Output-Text für den User (Konsole/GUI)
	SQLKartenRepo				Verwalten der Kartendaten mit Datenspeicher: Datenbank
	SQLRollenRepo				Verwalten der Kartendaten mit Datenspeicher: Datenbank
	SQLVerbindung (I)			Interface, dass Methoden zum Ausführen von SELECT-Querys aus einer DB enthält. (Speichern nicht vorgesehen)
	
Application
	GameLoop					Verwaltet Spielphasen
	Nacht						Bildet eine Spielphas ab
	LibraryInterface			--nicht verwendet--
	LibraryManager				Verknüpft Karten mit Rollen 
	Ruleset						--nicht verwendet-- / Verwaltet allgemeine Spielregeln?
	
Domain
	Karte						Bildet eine Spielkarte ab
	KartenRepository (I)		Interface zur Verwaltung (Erzeugung & Auslesen) von Karten
	Rolle						Bildet eine Spielrolle ab
	RollenRepository (I)		Interface zur Verwaltung (Erzeugung & Auslesen) von Rollen
	Rule						--nicht verwendet--
	Spieler						Bildet ein Spieler ab 
	GameException				Exceptions für den Gameloop
